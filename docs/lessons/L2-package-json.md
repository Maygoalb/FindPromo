## Start a Node.js project
In that terminal, type this and press Enter:
```
npm init -y
```

**What this does (plain words):**
- `npm` = a tool that comes with Node.js for managing project tools/add-ons (like an app store for code pieces)
- This command creates a small file called `package.json` — think of it as an "info card" for your project (name, what tools it uses, etc.)

You should see a new file `package.json` appear in your folder on the left sidebar.

## What is `package.json`?
This file is like an "info card" for the project. It lists:
- The project name and version
- Which packages (tools) the project needs to run
- Custom shortcut commands (called "scripts")
